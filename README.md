# Twitter/X Automation

Automated Twitter posting for @ebubechukwu_sam.

## Approach

Browser-based automation via PlayWriter (no API keys needed).
Logs into x.com in an existing Chrome session and posts tweets directly.

## Setup

1. Ensure PlayWriter extension is installed in Chrome
2. Run: `playwriter init`
3. Login to twitter.com manually or via PlayWriter script
